# 🚀 REX NASA APOD Viewer

A Python CLI app that fetches NASA's Astronomy Picture of the Day (APOD) live from NASA's official API. View today's space picture or search any date back to 1995.

---

## 🌌 Features
- 📅 View today's NASA astronomy picture
- 🔍 Search any date (YYYY-MM-DD format)
- 📝 Get full title, date, explanation and image URL
- ⚠️ Handles connection errors and invalid dates

---

## 🛠️ Requirements
- Python 3.x
- requests library

Install:
pip install requests

---

## 🔑 API Key
Get your free key at: https://api.nasa.gov
Replace YOUR_KEY in the code with your actual key.

---

## ▶️ How to Run
python main.py

---

## 📋 Menu
1. View today's picture — pulls live NASA data for today
2. View by date — enter any date from 1995 to today
3. Exit

---

## 📅 Date Format
Always use: YYYY-MM-DD
Example: 2025-10-07

---

## 🔗 API Used
NASA APOD API — https://api.nasa.gov
Endpoint: GET https://api.nasa.gov/planetary/apod?api_key=YOUR_KEY
